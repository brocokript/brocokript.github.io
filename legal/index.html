Condiciones de uso y límites de responsabilidad.

Este texto explica de forma directa qué hace el software, qué no puede garantizar y cómo contactar para dudas puntuales.

1. Alcance del software 2. Privacidad y datos 3. Responsabilidad del usuario 4. Garantías y soporte 5. Licencia y pagos 6. Contacto para dudas puntuales

1. Alcance del software

BrocoKript es un software de cifrado y descifrado de archivos para Windows. Su objetivo es ayudar al usuario a proteger información localmente.

2. Privacidad y datos

El flujo principal se ejecuta en el equipo del usuario. BrocoKript no necesita acceder al contenido de tus archivos para completar cifrado y descifrado en el uso normal.

3. Responsabilidad del usuario

  • El usuario es responsable de custodiar sus claves o contraseñas.
  • Si la clave se pierde, no existe recuperación de datos cifrados.
  • El usuario es responsable del uso legal del software y del contenido que protege.

4. Garantías y soporte

  • El software se entrega tal cual ("as is").
  • No se garantiza seguridad al 100%, porque ningún software puede garantizarlo.
  • No se ofrece asistencia personalizada ni resolución individual de incidencias.

5. Licencia y pagos

  • Community: versión gratuita con funciones y límites de uso básico.
  • Pro: pago único para desbloquear más comodidad, automatización y gestión avanzada.
  • La licencia habilita uso del software según la versión adquirida.
