Detalles técnicos de la aplicación.

Motor de cifrado

El motor principal de BrocoKript está desarrollado en Rust. Se eligió este lenguaje por su enfoque en seguridad de memoria y rendimiento estable.

El software utiliza cifrado AES-256-GCM para proteger archivos.

Ver código fuente del motor en Rust

Ejecución local

El cifrado y descifrado se ejecutan en el equipo del usuario. La aplicación no necesita enviar archivos para completar el flujo básico.

Flujo de operación

  • Selección del archivo o carpeta.
  • Ejecución de cifrado o descifrado con clave definida por el usuario.
  • Registro local de estado para revisar resultados.

Límites conocidos

  • La versión Community está orientada a uso manual y con límites operativos.
  • No existe recuperación de claves perdidas.
  • Ningún software puede garantizar seguridad al 100% en todos los contextos y entornos.
